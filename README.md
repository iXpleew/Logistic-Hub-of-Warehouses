# Logistic Hub of Warehouses
## Introduction to project
  Project was created as an assignent for "Fundamentals of Computer Science and Programming" course.
  The main topics I had to implement were:
    - showing warehouse's current capacity
    - showing actual requests
    - showing distrubution of a product around the hub
    - adding/removing products from warehouse
    - implemented dijkstra algorithm for searching the shortest paths
    - skipping time
    - loading and saving progress
  This application includes all of the above assumptions
